# JSON 预览助手 (Chrome 扩展)

一个轻量的 Chrome 浏览器扩展：点击图标打开**新标签页**，三栏布局——**左侧文件栏** | **中间编辑器** | **右侧预览**。

支持 **JSON 格式化/折叠/搜索** 和 **Markdown 实时渲染预览**，可新建、打开、保存文件到本地磁盘。

## 功能

### 文件管理
- 📁 **左侧文件栏**：列出所有已打开的文件，点击切换
- ➕ **新建文件**：支持新建 `.json` 和 `.md` 文件
- 📂 **打开本地文件**：使用 File System Access API 从磁盘读取
- 💾 **保存到本地**：`Ctrl/⌘ + S` 保存回原文件或另存为
- ✏️ **重命名**：双击文件名即可修改
- 🗑️ **删除**：悬浮显示删除按钮
- 💡 **记忆恢复**：关闭 tab 后重新打开，文件列表和内容自动恢复

### JSON 预览
- 🎨 语法高亮（key / string / number / boolean / null）
- 🌳 可折叠 / 展开的树形结构
- 🔍 关键字搜索，实时高亮匹配
- 🧰 格式化 / 压缩 / 复制
- 🩹 容错解析：自动修复尾随逗号、单引号、未加引号的 key

### Markdown 预览
- 📖 实时渲染（输入即刷新，300ms 防抖）
- 支持 GFM（表格、代码块、任务列表）
- 深色主题 typography 样式

### 界面
- 🪟 新标签页全屏布局
- 📐 三栏可伸缩：中间分隔线可拖动
- 🎨 深色主题

## 目录结构

```
json-preview-extension/
├── manifest.json          # MV3 清单 (permissions: storage)
├── background.js          # Service Worker: 点击图标 → 打开/聚焦 tab
├── tab/
│   ├── index.html         # 三栏布局页面
│   ├── index.css          # 深色主题 + 侧边栏 + MD渲染样式
│   ├── index.js           # 文件管理 + JSON渲染 + MD渲染 + 交互
│   └── lib/
│       └── marked.min.js  # Markdown 解析库 (marked v15.0.7)
├── icons/                 # 可选图标目录
└── README.md
```

## 安装（开发模式）

1. 打开 Chrome，访问 `chrome://extensions/`
2. 右上角开启 **开发者模式**
3. 点击 **加载已解压的扩展程序**
4. 选择本项目根目录 `json-preview-extension/`
5. 点击工具栏的扩展图标，新 tab 中打开 JSON 预览页面

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl/⌘ + Enter` | 格式化 JSON |
| `Ctrl/⌘ + S` | 保存文件到磁盘 |

## 技术栈

- Chrome Extension Manifest V3
- 原生 HTML / CSS / JS（零构建）
- [marked.js](https://github.com/markedjs/marked) v15.0.7 - Markdown 解析
- File System Access API - 本地文件读写
- chrome.storage.local - 状态持久化
